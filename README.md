# sara_fresh
sarafresh hompage with node.js by lee

- backend : node.js , ejs, express, routing
- deploy server : heroku + git
- frontend : sass, css3, html5, js
- design : sara food (actual company)

- SASS코드 파편화 : 디자인에 그리드 미사용, 폰트/색 일관성 떨어짐 -> 협의할 부분
- SASS코드 주석 / 모듈화 : 작성하기 전에 공통점 미리 찾고 설계했어야 함.
